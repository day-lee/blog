# Daylee Tech Blog

A personal tech blog built with [Hugo](https://gohugo.io/) and the [Monochrome](https://github.com/kaiiiz/hugo-theme-monochrome) theme.

## Getting Started

### Prerequisites

- [Hugo Extended](https://gohugo.io/installation/) (v0.146.0 or later)
- [Git](https://git-scm.com/)

### Local Development

```bash
# Clone the repository
git clone https://github.com/day-lee/daylee-tech-blog.git
cd daylee-tech-blog

# Install theme submodule
git submodule update --init --recursive

# Run local server
hugo server
